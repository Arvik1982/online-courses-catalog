# Online Courses Catalog

Каталог онлайн-курсов с поиском и фильтрацией по категориям.

## 🚀 Деплой

## 📋 Функционал

- Адаптивная вёрстка
- Поиск по названию курсов
- Фильтрация по категориям
- Pixel-perfect по макету Figma
- Декоративные элементы фона

## 🛠 Стек

- **HTML + Vanilla JS** (ES Modules)
- **SCSS** (переменные, миксины, БЭМ)
- **Vite** (сборка)
- **ESLint + Prettier** (форматирование кода)

## 🔧 Запуск локально

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Проверка кода
npm run lint
npm run format
```
